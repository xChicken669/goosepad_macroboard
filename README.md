# Goosepad

A custom 6-key macro pad powered by the Seeed XIAO RP2040 and running QMK Firmware.<br><br>

Goosepad is a compact DIY hackpad designed for shortcuts, macros, and custom controls. It features six individually wired keys, an OLED status display, and a custom PCB with a 3D printed case.
## Features

- 6 programmable keys<br>
- Seeed XIAO RP2040 controller<br>
- QMK Firmware support<br>
- 0.91" 128x32 OLED display<br>
- Custom PCB<br>
- Custom 3D printed case<br>
- Direct GPIO button wiring (no matrix)<br>
